# AI 분류 결과 UI 확인 가이드

## 📍 AI 분류 결과를 볼 수 있는 위치

AI 분류 결과는 **이슈 상세 패널 (Issue Detail Panel)**에서 확인할 수 있습니다.

### 접근 방법

1. **메인 대시보드** (`/`)에서 이슈 카드를 클릭
2. 화면 오른쪽에 이슈 상세 패널이 열립니다
3. 상세 패널의 **상단 섹션**에서 AI 분류 결과를 확인할 수 있습니다

### 표시 위치

```
┌─────────────────────────────────────┐
│ 이슈 상세 패널 (오른쪽 사이드바)      │
├─────────────────────────────────────┤
│ 소스 | 심각도 | 카테고리 | 생성 시각 │
│ 원문 링크                            │
│                                     │
│ ┌─────────────────────────────────┐ │
│ │ 🤖 AI 분류 [AI]                 │ │ ← 여기!
│ │                                 │ │
│ │ AI 분류 이유가 여기에 표시됩니다 │ │
│ └─────────────────────────────────┘ │
│                                     │
│ 상태 | 담당 에이전트                │
│ 코멘트 섹션                         │
└─────────────────────────────────────┘
```

## 🎨 UI 디자인

### AI 분류 결과 박스

- **배경색**: 보라색-파란색 그라데이션 (`from-purple-50 to-blue-50`)
- **테두리**: 보라색 (`border-purple-200`)
- **위치**: 이슈 기본 정보(소스, 심각도 등) 아래, 상태/담당자 섹션 위

### 표시 내용

1. **분류 방법 배지**:
   - `🤖 AI 분류` (AI로 분류된 경우)
   - `📋 규칙 기반 분류` (규칙으로 분류된 경우)
   - AI 분류인 경우 추가로 `AI` 배지 표시

2. **분류 이유**:
   - `aiClassificationReason` 필드에 저장된 AI의 분류 근거가 표시됩니다
   - 예: "이 이슈는 게임 크래시와 관련된 것으로 보이며, 심각도 1로 분류됩니다."

3. **분류 이유가 없는 경우**:
   - AI 분류: "AI가 이 이슈를 자동으로 분류했습니다."
   - 규칙 분류: "키워드 기반 규칙으로 분류되었습니다."

## 📋 코드 위치

- **AI 분류**: `backend/services/aiIssueClassifier.js`

## 🔍 확인 방법

1. **메인 대시보드 접속**
   - 브라우저에서 `http://localhost:5173` (또는 프론트엔드 포트) 접속
   - 로그인 후 메인 대시보드로 이동

2. **이슈 선택**
   - 이슈 목록에서 AI로 분류된 이슈를 찾습니다
   - 이슈 카드를 클릭합니다

3. **AI 분류 결과 확인**
   - 오른쪽에 열리는 상세 패널에서 보라색/파란색 박스를 찾습니다
   - AI 분류 방법과 이유를 확인합니다

## ⚠️ 주의사항

- AI 분류 결과는 **AI API 키가 설정되어 있고**, AI 분류가 성공한 경우에만 표시됩니다
- 규칙 기반 분류인 경우에도 분류 방법은 표시되지만, AI 분류 이유는 없을 수 있습니다
- `aiClassificationMethod`가 `null`이거나 없는 경우, AI 분류 결과 박스가 표시되지 않습니다

## 🛠️ 문제 해결

### AI 분류 결과가 표시되지 않는 경우

1. **백엔드 확인**:
   - `OPENAI_API_KEY` 환경 변수가 설정되어 있는지 확인
   - 백엔드 로그에서 AI 분류 오류 확인

2. **데이터베이스 확인**:
   ```sql
   SELECT id, summary, aiClassificationMethod, aiClassificationReason 
   FROM ReportItemIssue 
   WHERE aiClassificationMethod IS NOT NULL 
   LIMIT 10;
   ```

3. **프론트엔드 확인**:
   - 브라우저 개발자 도구 콘솔에서 에러 확인
   - 네트워크 탭에서 API 응답 확인

### AI 분류 이유가 비어있는 경우

- AI API가 분류 이유를 반환하지 않은 경우
- 규칙 기반 분류인 경우 (AI 분류 이유 없음)
- 이는 정상 동작입니다
