- **작성 시간**: 15:25 (오늘)

## 근본 원인 발견

### 문제점: 게시판 이름 추출 실패

**위치**: `backend/workers/monitoring/naverCafe.worker.js` 863줄

```javascript
if (text && text.length < 20 && !text.match(/^\d+$/) && !text.match(/^\d{4}-\d{2}-\d{2}/)) {
  boardNameElement = cell;
  break;
}
```

**문제**:
- 게시판 이름이 **20자 이상**이면 추출 실패
- "PUBG PC 질문/응답 게시판"은 13자이므로 이 조건은 통과해야 함
- 하지만 게시판 이름 추출이 실패하면 `boardName`이 빈 문자열이 됨

### 게시판 이름 추출 실패 시 동작

1. **게시판 이름 추출 실패** → `boardName = ''`
2. **정규화** → `normBoardName = ''`
3. **제외 게시판 필터링**:
   ```javascript
   if (normBoardName && Array.isArray(excludedBoards) && excludedBoards.length > 0) {
     // 이 조건이 false가 되어 필터링을 건너뜀
   }
   ```
   - `normBoardName`이 빈 문자열이면 제외 게시판 필터링을 통과함
   - 즉, 게시판 이름 추출 실패 시 제외 게시판 필터링을 우회함

### 실제 원인 추정

게시판 이름 추출이 실패한 이유는 다음 중 하나일 가능성이 높습니다:

1. **HTML 구조 변경**: 네이버 카페 리스트 페이지의 HTML 구조가 변경되어 게시판 이름을 추출하지 못함
2. **선택자 실패**: 모든 게시판 이름 선택자가 실패
   - `a.board_name, .board_name`
   - `td.td_board a, td.td_category a`
   - `.board_area .board_name, .category_name`
   - 테이블 셀에서 직접 추출 (20자 제한)

3. **게시판 이름이 다른 형식으로 표시**: 
   - 예: "질문/응답"만 표시되고 "PUBG PC"는 생략
   - 예: 게시판 이름이 다른 위치에 있음

### 해결 방안

1. **게시판 이름 추출 로직 개선**
   - 20자 제한 완화 또는 제거
   - 더 많은 선택자 추가
   - 게시판 이름 추출 실패 시 로그 강화

2. **게시판 이름 추출 실패 시 처리**
   - 게시판 이름 추출 실패 시에도 게시글을 수집하도록 변경
   - 또는 게시판 이름 추출 실패를 명시적으로 로깅

3. **디버깅 강화**
   - 게시판 이름 추출 실패 시 상세 로그 출력
   - HTML 구조 스냅샷 저장
