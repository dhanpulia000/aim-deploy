# "컴린이 질문드립니다" 게시글 수집 실패 원인 분석

## 분석 일시
2025-12-19

## 확인된 사항

### 1. 키워드 필터링 ✅
- **상태**: 통과
- **이유**: 모니터링 키워드가 설정되어 있지 않아 모든 게시글이 통과
- **결론**: 키워드 필터링이 원인 아님

### 2. 제외 게시판 필터링 ⚠️
- **설정된 제외 게시판**:
  - IT 정보 공유
  - 펍지 코디 게시판
  - 펍지 FUN SHOT
  - 펍지 FUN VIDEO
  - 듀오/스쿼드
  - 클랜/방송/디스코드
  - RP 보너스 파티 모집
  - 듀오/스쿼드 모집
  - 클랜 홍보
  - 오픈 채팅방 홍보
  - 스크림팀 홍보/구인/구직
  - 인기도 교환
  - 로비 앨범 교환
- **기본 제외 게시판** (설정 없을 때):
  - 가입인사
  - 등업신청
  - 자유게시판
- **필터링 로직**: 양방향 부분 일치 (게시판 이름에 제외 키워드가 포함되거나, 제외 키워드에 게시판 이름이 포함되면 제외)
- **확인 필요**: 해당 게시글이 위 게시판 중 하나에 있는지 확인 필요

### 3. 날짜/시간 필터링 ⚠️
- **필터링 조건**:
  - `lastScanAt` 이후에 작성된 게시글만 수집
  - 또는 `lastArticleId`보다 큰 articleId를 가진 게시글만 수집
- **확인 필요**: 게시글 작성 시점이 마지막 스캔 시간보다 이전인지 확인 필요

### 4. 게시판 이름 추출 실패 가능성 ⚠️
- **가능성**: 리스트 페이지에서 게시판 이름을 제대로 추출하지 못했을 수 있음
- **영향**: 게시판 이름 추출 실패 시 제외 게시판 필터링이 제대로 작동하지 않을 수 있음

## 가능한 원인 (우선순위 순)

### 1순위: 제외 게시판 필터링
- 해당 게시글이 제외 게시판 목록에 있는 게시판에 작성되었을 가능성
- 특히 "질문" 관련 게시판이 제외 목록에 포함되어 있는지 확인 필요

### 2순위: 날짜/시간 필터링
- 게시글이 마지막 스캔 시간보다 이전에 작성되었을 가능성
- 또는 articleId가 마지막 articleId보다 작을 가능성

### 3순위: 게시판 이름 추출 실패
- 리스트 페이지의 HTML 구조가 변경되어 게시판 이름을 추출하지 못했을 가능성
- 이 경우 제외 게시판 필터링이 제대로 작동하지 않을 수 있음

## 확인 방법

1. **게시글 위치 확인**
   - 해당 게시글이 어떤 게시판에 있는지 확인
   - 게시판 이름이 제외 목록에 포함되어 있는지 확인

2. **게시글 작성 시점 확인**
   - 게시글 작성 날짜/시간 확인
   - 마지막 스캔 시간과 비교

3. **로그 확인**
   - 크롤링 로그에서 해당 게시글이 스킵되었는지 확인
   - 스킵 사유 확인 (제외 게시판, 날짜 필터링 등)

## 권장 조치

1. 해당 게시글의 URL과 게시판 이름을 확인
2. 크롤링 로그에서 해당 게시글 관련 로그 검색
3. 필요시 제외 게시판 목록 조정 또는 게시판 이름 추출 로직 개선
